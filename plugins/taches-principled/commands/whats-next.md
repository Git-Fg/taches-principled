Create a handoff document at whats-next.md capturing everything needed to continue this work. Include the original task scope, files created or modified, key decisions, remaining steps with file paths and targets, blockers encountered, critical context and assumptions, and current state. Prioritize precision over brevity — the goal is zero information loss for the next session.
